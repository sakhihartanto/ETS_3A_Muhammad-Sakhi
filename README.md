## Installation Front-end

Untuk project setup Vue maka coba ke folder : 
```bash
cd client 
```
Lalu Lakukan : 
```bash
npm install
```

## Run It !
### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
